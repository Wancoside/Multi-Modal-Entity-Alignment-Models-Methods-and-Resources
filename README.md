# Multi-Modal Entity Alignment (MMEA) Models Collection

## Introduction

**Multi-Modal Entity Alignment (MMEA)** is a crucial task in Knowledge Graph (KG) integration that aims to identify equivalent entities across different KGs by leveraging various modalities, primarily structured information (triples) and non-structural context (such as images, text descriptions, and attributes). This repository compiles a collection of state-of-the-art and influential MMEA models and their essential details.

## 2. Key Datasets & Benchmarks

MMEA models are commonly evaluated on benchmark datasets derived from DBPedia, Freebase, Wikidata, and YAGO.

| Dataset Name      | Source KGs Involved     | Alignment Type | Common Subsets           | Notes                                                                                 |
|:------------------|:------------------------|:---------------|:-------------------------|:--------------------------------------------------------------------------------------|
| **DBP15K**        | DBPedia (Multi-lingual) | Cross-lingual  | ZH-EN, FR-EN, JA-EN      | Standard benchmark for multilingual EA.                                               |
| **FB15K-DB15K**   | Freebase & DBPedia      | Heterogeneous  | N/A                      | Alignment between two large, structurally different KGs.                              |
| **FB15K-YAGO15K** | Freebase & YAGO         | Heterogeneous  | N/A                      | Alignment between Freebase and YAGO.                                                  |
| **DWY15K**        | DBPedia, Wikidata, YAGO | Multi-Source   | DBP-WD(N/D), DBP-YG(N/D) | A more challenging benchmark often divided into Normal (N) and Difficult (D) subsets. |

## 3. Evaluation Metrics

MMEA task performance is primarily measured using the following metrics:

| Metric                         | Description                                                                                                                       | Optimal Value   |
|:-------------------------------|:----------------------------------------------------------------------------------------------------------------------------------|:----------------|
| **Hits@N**                     | The ratio of correctly aligned entities that appear in the top $N$ prediction candidates. **Hits@1** is the most widely reported. | 100\%           |
| **MRR (Mean Reciprocal Rank)** | The average of the reciprocal ranks of the correct alignments. Higher MRR is better.                                              | 1.0             |
| **Mean Rank**                  | The average rank of the correct alignment in the candidate list.                                                                  | Lower is better |
